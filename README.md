
# Launchpad plasma

A fullscreen menu (plasmoid) for kde plasma :blue_heart: 

### Features
- Show/hide filter list
- Hide applications
- Favorites apps 

## Installation

Download the menu from: [https://store.kde.org/p/1364064/](https://store.kde.org/p/1364064/)
